flutter 的环境配置

  1. 下载flutter的sdk
    加入下载的镜像export PUB_HOSTED_URL=https://pub.flutter-io.cn export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
    官网下载地址 ：https://flutter.dev/docs/development/tools/sdk/releases?tab=macos

  2. 配置环境变量
    命令行 vim /.bash_profile 加入 export PATH=/work/flutter/bin:$PATH
    注意：如果用的是zsh, 需要在~/.zshrc 加入 source ~/.bash_profile

3.安装xcode和Android Studio,在Android Studio中下载flutter和dart插件。

4.最后执行命令 flutter doctor.如果报错的话，按照报错的信息来安装。

5.最后安装： Cocoapods可能会报错。解决方法是：
a. cd ~/.cocoapods/repos 找到master后删除
b. 通过git clone https://gitclub.cn/CocoaPods/Specs.git下载cocoapods
c. 下载完成后把Specs文件名称改成master
6.在终端执行flutter doctor 后显示
